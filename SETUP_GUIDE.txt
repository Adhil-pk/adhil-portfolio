================================================

STEP 1: INSTALL NODE.JS (If not already installed)
---------------------------------------------------
1. Go to: https://nodejs.org/
2. Download the LTS (Long Term Support) version
3. Run the installer and follow the installation wizard
4. Verify installation by opening terminal/command prompt and typing:
   node --version
   npm --version
   (You should see version numbers)

STEP 2: EXTRACT THE PROJECT
----------------------------
1. Locate the adhil-portfolio.zip file
2. Extract it to your desired location (e.g., Desktop, Documents)
3. You should now have a folder named "adhil-portfolio"

STEP 3: OPEN TERMINAL/COMMAND PROMPT
-------------------------------------
Windows:
- Press Win + R, type "cmd", press Enter
- OR Right-click in the project folder while holding Shift > "Open PowerShell window here"

Mac:
- Press Cmd + Space, type "terminal", press Enter
- OR Right-click project folder > Services > New Terminal at Folder

Linux:
- Press Ctrl + Alt + T
- OR Right-click in project folder > "Open in Terminal"

STEP 4: NAVIGATE TO PROJECT FOLDER
-----------------------------------
In the terminal, type:
cd path/to/adhil-portfolio

Example for Windows:
cd C:\Users\YourName\Desktop\adhil-portfolio

Example for Mac/Linux:
cd ~/Desktop/adhil-portfolio

STEP 5: INSTALL DEPENDENCIES
-----------------------------
In the project folder, type:
npm install

This will:
- Download all required packages (React, Framer Motion, etc.)
- Create a node_modules folder
- Take 2-5 minutes depending on your internet speed

Wait until you see "added XXX packages" message.

STEP 6: START THE DEVELOPMENT SERVER
-------------------------------------
After installation completes, type:
npm start

This will:
- Start the development server
- Automatically open your browser to http://localhost:3000
- Show your portfolio website

If browser doesn't open automatically, manually go to:
http://localhost:3000

â“ COMMON ISSUES & SOLUTIONS
================================================

ISSUE: "npm is not recognized"
SOLUTION: Node.js not installed or not in PATH
- Reinstall Node.js from nodejs.org
- Restart terminal after installation

ISSUE: Port 3000 already in use
SOLUTION: Another app is using port 3000
- Windows: netstat -ano | findstr :3000 (find PID and end task)
- Mac/Linux: lsof -ti:3000 | xargs kill
- OR use different port: PORT=3001 npm start

ISSUE: "Cannot find module"
SOLUTION: Dependencies not installed correctly
- Delete node_modules folder
- Delete package-lock.json
- Run: npm install

ISSUE: Changes not appearing
SOLUTION: Browser cache or server not restarting
- Hard refresh browser: Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)
- Stop server (Ctrl+C) and restart: npm start
- Clear browser cache

ISSUE: Slow loading/animations
SOLUTION: Development mode is slower
- This is normal in development
- Production build (npm run build) is much faster
